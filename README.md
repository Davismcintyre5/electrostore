# ElectroStore Customer Frontend

React + Vite customer storefront for ElectroStore e-commerce platform.

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and set `VITE_API_URL`
3. Run dev server: `npm run dev`

## Features

- Browse products with filtering and search
- Product details
- Shopping cart
- Checkout with address selection
- User authentication (login/register)
- Account management (profile, password)
- Address book (max 3 addresses, set default)
- Wishlist
- Order history and invoice printing
- Responsive design with Tailwind CSS

## Build

`npm run build` – outputs to `dist` folder.
